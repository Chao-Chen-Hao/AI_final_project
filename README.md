# AI_final_project

### Classifier
- Model: EfficientNet-b4 (ICML'19: https://arxiv.org/pdf/1905.11946.pdf)
- Training Set: 3000 images for each class
- validation Set: 500 images for each class
- Performance: (Accuracy ~= 95% on the validation set)
